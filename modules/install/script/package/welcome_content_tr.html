The XE workspace package allows user to build a workspace site directly, in which they can manage and trace the projects&communities that they are working on.

The objective of the workspace is to create a well-controlled project system,  which can improve the working efficiency.


Workspace refer to small premisies provided, often by local authorities or economic development ogencies, to help new businesses to establish themselves. These typically provide not only physical space and utilities, but also administrative services and links to support and finance organisations, as well as peer support among the tenants.
