CafeXE 설치를 환영합니다!

CafeXE설치가 성공적으로 완료되었습니다.
이 카페는 데모 카페 이므로 추후 사이트 운영시 삭제 가능합니다.
관리자 권한으로 로그인하면 설치된 모든 요소를 확인할 수 있습니다.

  1. 헤더, 본문, 풋터로 구성된 화면이 보인다면 레이아웃이 정상적으로 설치된 것입니다.
  2. 통합 검색 입력창이 보인다면 통합검색 모듈이 설치된 것입니다.
  3. 로그인 정보, 카페 메뉴, 방문자 정보가 보인다면 위젯이 설치된 것입니다.
  4. 이 메시지가 보인다면 게시판 모듈이 설치된 것입니다.

CafeXE와 관련된 설정은 CafeXE관리자 화면에서 변경 가능합니다.
